How to connect Gmail:
1. Get a Google OAuth token at OAuth Playground
2. Select Gmail API v1 scopes: gmail.send + gmail.readonly
3. Exchange for tokens → copy Access Token + Refresh Token
4. Set in Railway: GOOGLE_ACCESS_TOKEN and GOOGLE_REFRESH_TOKEN
5. Or POST to /api/auth/token with {"{"}access_token, refresh_token{"}"}
